# Hugeicons Free — Stroke Rounded

The SVG assets in `src/SyncClipboard.Core/Assets/EmbeddedIcons/` are selected from
the **Hugeicons Free — Stroke Rounded** icon set. They are embedded in SyncClipboard
and used as content-type indicators.

- Source package: [`@hugeicons/core-free-icons`](https://www.npmjs.com/package/@hugeicons/core-free-icons), version 4.2.2
- Upstream package license: [MIT](https://www.npmjs.com/package/@hugeicons/core-free-icons?activeTab=readme)
- Icon provider: [Hugeicons](https://hugeicons.com/)
- Upstream usage and licensing FAQ: [Hugeicons licensing FAQ](https://hugeicons.com/faq/general/license)

Only the selected SVG assets required by SyncClipboard are included here; this is not a
copy of the complete Hugeicons icon set. When updating or reusing these assets, check
the upstream package license and Hugeicons terms in effect at that time.
